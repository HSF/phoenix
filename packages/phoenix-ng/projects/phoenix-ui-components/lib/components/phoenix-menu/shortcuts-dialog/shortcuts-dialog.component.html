Keyboard Shortcuts

Event & Session Controls

Next event Shift + ArrowRight
Previous event Shift + ArrowLeft
Toggle session recording Shift + S

Camera & Viewport Controls

Toggle auto-rotation Shift + R
Zoom in Shift + +
Zoom out Shift + -
Revert main camera Shift + V
Toggle local clipping Shift + C

UI & Theming Controls

Toggle dark theme Shift + T
Preset views (if configured) Shift + 1-9
